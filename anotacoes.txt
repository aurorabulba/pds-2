Como compilar manualmente?

1. g++ -c nome-do-arquivo.cpp 
    Se o programa tiver multiplos arquivos, fazer isso para todos
    Irá gerar um arquivo nome-do-arquivo.o
2. g++ -o nome-do-arquivo.exe nome-do-arquivo.o
    Vai gerar o executável do programa
3. & .\'main.exe'
    Executar o programa

Como fazer um commit no GitHub?

1. git add .
2. git push origin main
3. git commit -m "nome-da-pasta"
